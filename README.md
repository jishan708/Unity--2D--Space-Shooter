# Unity 2D Space Shooter

A 2D arcade-style space shooter game developed in Unity using C#.

## Features

* Player movement
* Enemy spawning system
* Collision detection
* Score tracking
* Game Over screen
* Restart functionality

## Controls

| Key             | Action     |
| --------------- | ---------- |
| A / Left Arrow  | Move Left  |
| D / Right Arrow | Move Right |

## Technologies Used

* Unity
* C#

## Project Structure

* Assets
* Packages
* ProjectSettings
